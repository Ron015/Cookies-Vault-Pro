# 🍪 Cookie Vault Pro — Chrome Extension

> ⚡ Multi-Account Cookie Manager • Local + Cloud Sync • Ultra Fast

---

## 🚀 Overview

Cookie Vault Pro is a powerful Chrome Extension that lets you capture, store, switch, and sync cookies instantly across multiple accounts and domains.

Perfect for:
- 🧪 Testing multiple sessions
- 👨‍💻 Developers & QA
- 🔐 Security / Ethical workflows
- 🎯 Managing multiple logins

---

## ✨ Features

### 🍪 Cookie Management
- Capture current website cookies
- Switch between saved sessions instantly
- Clean / clear cookies per domain
- Store multiple profiles per site

### ☁️ Cloud Sync (Firebase)
- Sync cookies across devices
- Hybrid storage (Local + Cloud)
- One-click sync
- Secure authentication

### 🧠 Smart UI
- Live search profiles
- Domain-specific filtering
- Global view (all cookies)
- Profile stats & badges

### 📁 Import / Export
- Export cookies as JSON
- Import backups anytime

## ⚙️ Installation

1. Clone repo:
   git clone https://github.com/ron015/cookies-vault-pro.git

2. Open Chrome:
   chrome://extensions/

3. Enable Developer Mode

4. Click Load Unpacked

5. Select project folder

---

## 🔥 Usage

- Capture cookies → Click Capture
- Switch profile → Select + Apply
- Enable cloud → Add Firebase config
